The War Thunder API

Serving less than 1 call each month!

All the War Thunder aircraft data you'll ever need in one place, easily accessible through a modern free open-source RESTful API.

Try it now!

http://localhost:5000/api/

Need a hint? Try these endpoints:

  • vehicles/aviation — List all aircraft
  • vehicles/aviation/aircraft/:id — Aircraft details
  • vehicles/aviation/helicopters — List helicopters
  • nations — List all playable nations

What is this?

This is a RESTful API linked to a comprehensive database detailing everything about the War Thunder series — aircraft, vehicles, and weapons.

Where do I start?

You can directly query this API using any HTTP client or browser. It's designed to be public, simple, and fast.
